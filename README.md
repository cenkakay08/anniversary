## Features

- **Countdown/Duration Tracking**: Calculates the time (Years, Months, Days, Hours, Minutes, Seconds) since or until January 23, 2025.
- **Dynamic Interface**: `CountUp` animations and live counter.
- **Visual Effects**:
  - Fireworks background (`FireworksBackground`).
  - Spring header animations (`SpringHeader`).
  - Heart animations.
- **Technologies**: Next.js, React, TypeScript, Tailwind CSS.

## Installation

To start the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

You can view the result by opening [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `pages/index.tsx`: Main page.
- `components/`: Application components (`Duration`, `WelcomeText`, etc.).
- `styles/`: Global styles.
